OCTAVE Statement

Sets the octave in which next notes will be played. See What is Note, Octave and bla bla bla

Syntax

OCTAVE:<Octave number>

The OCTAVE statement syntax has these arguments

Part Description
Octave number Octave number

Short Form

O

Remarks

This value is used only if you do not specify the octave as the part of the instruction. Default value is 4. You can specify values as high as 256 but it produces some unpredictable effects.

Examples

1. Set octave to 2 and then 6

OCTAVE:2
a b c
OCTAVE:6
a b c

2. Play notes in decreasing octave

SET:Count,7
LABEL:TheLoop
OCTAVE:Count
c# a a
MATH:Count,--
IF:Count,>,0,TheLoop
