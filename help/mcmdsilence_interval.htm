SILENCE_INTERVAL Statement

What it does: Sets the note interval value that will be used if no note interval is specified explicitly in the note commands.

Syntax

SILENCE_INTERVAL:<interval value>

The SILENCE_INTERVAL statement syntax has these arguments

Part Description
interval value One interval is by default equals to 60ms

Short Form

SI

Remarks

Default value is 1. Silence could be also inserted by SILENCE command. The value specified by SILENCE_INTERVAL command is used only if no such value is supplied as the part of instruction to play the note.

Examples

1. Silence interval demo

c#  a# 'Play the note followed by silence for default interval=4

SILENCE_INTERVAL:8
c#  a# 'Play the note followed by silence for new default interval=8

c#:,3 a# 'Play the note followed by silence for interval=3
