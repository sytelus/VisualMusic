WAIT_FOR Statement

What it does: Waits for specified amount of time before next instruction is executed.

Syntax

WAIT_FOR:<interval value>

The WAIT_FOR statement syntax has these arguments

Part Description
interval value One interval is by default equals to 60ms

Short Form

WF

Remarks

Default value is 1. This command is exactly same as SILENCE command.

Examples




c#
WAIT_FOR:10
a#

'Exactly same thing

c#
SILENCE:10
a#

'Exactly same thing

c#:,10
a#
