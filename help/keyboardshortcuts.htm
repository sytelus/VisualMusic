Keyboard Shortcuts

    1. Play notes: All keys in the row from "Q" to "]"

    2. Play tune: F5

    3. Compile the tune: Ctrl + F5

    4. Go to Instrument Selector: Alt + I

    5. Go to Tune List: Alt + T

    6. Go to Tune Editor: Alt + S

    7. Switch Octave: Alt + (1,2,...,9,A,B,C)

    8. Switch to volume change: Alt + V

    9. Switch to pan change: Alt + P

    10. Stop Play/recording: Ctrl + T

    11. Add new Tune Alt + '+'

    12. Rename Tune Alt + '~'

    13. Delete the Tune Alt + 'X'
