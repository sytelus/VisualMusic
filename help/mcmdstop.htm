STOP Statement

Stops playing another tune or itself.

Syntax

STOP:[Tune Name]

The STOP statement syntax has these arguments

Part Description
Tune Name Tune that needs to be stopped. If no Tune Name specified then currently running tune is stoped (which is same as EXIT). Tune name must exist is specified.

Short Form

None

Remarks

This instruction allows you stop other running tunes or itself.

Examples

1. Write in Tune1 following:

NOTE_INTERVAL:5
a b a a# b
PLAY:Tune2
a a# c
STOP:Tune2 'Tune2 execution terminated
c# d a a a#
STOP 'Execution is stoped here
c d d#

2. Write in Tune2 following:

NOTE_INTERVAL:5
INSTRUMENT:Sitar
a b a a# b a a# c c# d a a a# c d d#

See Also
