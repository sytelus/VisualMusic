LABEL


Command Format:

LABEL:<Label name>

What it does:

Labels the location in code where you can jump later.

Short Form:

  L

Parameters:

Label name: Name of label you want to give

Value constraints:

Label name: Can not contain space. You can't have two labels with same name.

Remarks:

This instruction use used to give a name to the location where you can jump later using JUMP instruction or IF instruction.

Examples:

1. Play the note c# again and again

LABEL:DoItAgain
c#
JUMP:DoItAgain

2. Play the notes c# 5 times

SET:Count,5
LABEL:TheLoop
c#
MATH:Count,--
IF:Count,>,0,TheLoop
