MATH Statement

Does the mathematical operation on the variable value

Syntax

MATH:<Variable Name>,<Operation>,[Parameters for operation]

The MATH statement syntax has these arguments

Part Description
Variable Name Name of the variable to which value will be used in operation
Operation Mathematical operation
++: Increment variable's value
--: Decrement variable's value
+=: Add the value specified in 3rd
parameter to the variable
-=: Subtract the value specified in
3rd parameter to the variable
*=: Multiply the value specified in
3rd parameter to the variable
/=: Divide the variable's value by
the value specified in 3rd parameter
^=: Raise the variable's value by
the value specified in 3rd parameter
Parameters for operation Additional parameter that is required for some of the above operations

Short Form

M

Remarks

This instruction is very basic one which allows you to some arithmetic with variables value. It doesn't support expressions like (2*3/4) etc.

Examples

1.

SET:Count,28
LABEL:TheLoop
INSTRUMENT:Count
c# a a b a#
MATH:Count,++
IF:Count,<,128,TheLoop
