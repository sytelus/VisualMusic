Extending Visual Music

Visual Music provides you several way to extend itself.

  • You can write Add-Ins for Visual Music. Add-ins are also called plug-ins which integrates itself with target software and adds more functionality to it.
  • You can script the Visual Music Object model using VBA or VBScript.
  • You can write your DDE client to control Visual Music. DDE is the way to talk to the running application. Visual Music is also a DDE server.
  • You can run Visual Music with command line parameters.
  • Finally Visual Music source code is available upon request. You too can take part in the Visual Music development.
