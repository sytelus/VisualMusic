PRINT  Statement

Prints the value of specified variable or parameter in the debug window.

Syntax

PRINT:<Variable or parameter name>,[Tag Word]

The  PRINT statement syntax has these arguments

Part Description
Variable or parameter name

This could be the name of the variable whoops value is needed to print or any of the following strings:

NI - to print the value of current Note  Interval
SI - to print the value of current silence  Interval
OCTAVE - to print the value of current octave
VOLUME - to print the value of current volume settings
PAN - to print the value of current pan settings

NOTE: The parameter name above must be in upper case.
Tag Word This is simply any string without space that you want to print along with the value

Short Form

None

Remarks

This instruction is vary useful for debugging purposes when you want to know what is the value of a variable or some MIDI parameter at any point in the script. The PRINT statement prints a line with following format:

Name_of_tune : Tag_Word, value

Examples

1.
SET:MyVar
RANDOM:MyVar,1000
PRINT:MyVar,Some_Random_Value

2.
SET:MyVar
RANDOM:MyVar
PRINT:MyVar
