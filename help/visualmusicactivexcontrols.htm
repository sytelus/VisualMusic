Visual Music ActiveX Controls

Visual Music also contains many ActiveX controls to simplify your task of creating your own UI. As with Visual Music objects, these ActiveX controls are currently compiled in to same EXE instead of exposing through OCX files to prevent version conflicts in final version.

Following is the short descriptions of each of the ActiveX control available:

  • InstrumentSelector : This control allows the user to select an instrument from the available list. It also provides the way to mark an instrument as your favourite, do a search and play the sample while selecting it.
  • MIDIKeyboard: This is the set of white and black keys that gives Piano like looks. This control generates event when you press one of the piano key. It also manages key stroke translation.
  • MIDIParamControls : This is a simple control with facility to select an octave, volume and pan.
  • MScriptEditor: This controls allows you to type the script for the tune, create new tunes, manage them, automatically save script for the tune on lost focus and much more!
