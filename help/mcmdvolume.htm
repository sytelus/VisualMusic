VOLUME Statement

What it does: Sets the amount of loudness with which next notes would be played

Syntax

VOLUME:<Volume number>

The   VOLUME statement syntax has these arguments

Part Description
Volume number Volume number: 0-100. Higher values could be specified but not recommended.

Short Form

V

Remarks

Default value is 100 (i.e. full). You can specify values as high as 255 but it produces some unpredictable effects.

Examples

1. Play with full volume

VOLUME:100
a b c

2. Play with half volume

VOLUME:50
a b c

3. Play with decreasing volume

SET:Count,100
LABEL:TheLoop
VOLUME:Count
c# a a
MATH:Count,--
IF:Count,>,0,TheLoop
