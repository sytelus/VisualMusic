JUMP


Command Format:

JUMP:<Label name>

What it does:

Jumps to the immediate next location where the specified label is there.

Short Form:

  J

Parameters:

Label name: Name of label where you want to jump

Value constraints:

Label name: Can not contain space. Label must exist or you will get run time error.

Remarks:

Use JUMP instruction to repeat particular commands again and again.

Examples:

Examples: 1. Play the note c# again and again

LABEL:DoItAgain
c#
JUMP:DoItAgain
