Using MScript Editor



Using MScript editor is easy. You always write a script for a tune. Each tune has some name. A tune is like a function or procedure in normal programming terminology. MScript editor allows you to choose the tune for which you want to write the script. Once you write the script you can press Play button to play it. To know which button is for what, put the mouse over that button for a while.

You can add more tunes by pressing on the "+" sign at the bottom of the list of tunes. To delete the tune press "X" sign. To rename the tune press "~" sign.

To clear the script box, press "X" sign on the right lower corner of it.
