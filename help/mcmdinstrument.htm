INSTRUMENT Statement

What it does: Sets the amount of loudness with which next notes would be played

Syntax

INSTRUMENT:<Instrument name or number>

The INSTRUMENT statement syntax has these arguments

Part Description
Instrument name or number Name or number of
the instrument.

Short Form

I

Remarks

Prefer to specify the instrument to play by it's name. You can drag and drop instrument from Instrument Selector in the Visual Music.

Examples

1. Set Music Box instrument

INSTRUMENT:Music_Box
a b c

2. Set instrument by number

INSTRUMENT:118
a b c

3. Set instrument from variable

SET:Count,28
LABEL:TheLoop
INSTRUMENT:Count
c# a a b a#
MATH:Count,++
IF:Count,<,128,TheLoop
