IF Statement

Compares the value of the variable with the specified value and jumps accordingly to the specified label

Syntax

IF:<Variable Name>,<Comparison Condition>,<Value to compare with>,<Label to jump on if condition is true>, [Label to jump on if condition is false]

The MATH statement syntax has these arguments

Part Description
Variable Name Name of the variable whose value is to be compared
Operation Any of the comparison operation >, <, >=, <=, ==, <>
Value to compare with A literal value which is to be compared with variable's value
Label to jump on if condition is true If condition evaluates to true the jump is made to this label
Label to jump on if condition is false If condition evaluates to false the jump is made to this label. Note that this
is optional argument.

Short Form

None

Remarks

This is the basic IF structure that you can use to make decision of where to divert the program flow.

Examples

1.

'Play the notes c# 5 times

SET:Count,5
LABEL:TheLoop
c#
MATH:Count,--
IF:Count,>,0,TheLoop
