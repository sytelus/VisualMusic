NOTE_INTERVAL Statement

Sets the amount of time for which the note will be played if no such time is specified.

Syntax

NOTE_INTERVAL:<interval value>

The NOTE_INTERVAL statement syntax has these arguments

Part Description
interval value One interval is by default equals to 60ms

Short Form

NI

Remarks

Default value is 1. The value specified by NOTE_INTERVAL command is used only if no such value is supplied as the part of instruction to play the note.

Examples

1. NOTE_INTERVAL demonstration

c# 'Play note for default interval=1
NOTE_INTERVAL:8
c# 'Play note for new default interval=8
c#:3 'Play note for interval=3
