RANDOM Statement

Sets the specified variable with the randomly generated integer.

Syntax

RANDOM:<Variable name>,[Upper Limit]

The RANDOM statement syntax has these arguments

Part Description
Variable Name Name of the variable in which random number will be stored.
Upper Limit Upper limit of the generated number (including this number). Default is 100.

Short Form

None

Remarks

This instruction is useful you want to play random tunes or set random octave etc.

Examples

1.
SET:NoteNum
LABEL:TheLoop
RANDOM:NoteNum,88
NOTE:NoteNum,3,0
IF:Count,<,88,TheLoop
