SET Statement

Set the value of the variable to the specified value.

Syntax

SET:<Variable Name>,[Variable Value]

The SET statement syntax has these arguments

Part Description
Variable Name Name of the variable needed to be created or whose value is needed to be set.
Variable Value Value with which variable is to be set.

Short Form

None

Remarks

A variable is an entity which may contain some value that you can read anytime or if you want you can set it's value to something else. SET statement creates a variable if it didn't existed. You must use SET first if you want to use variable.

Examples

1.

SET:NoteVal,52
NOTE:NoteVal

SET:NoteVal,++ 'Increment by one
NOTE:NoteVal 'Play this note


2.

SET:Count,28
LABEL:TheLoop
INSTRUMENT:Count
c# a a b a#
MATH:Count,++
IF:Count,<,128,TheLoop
