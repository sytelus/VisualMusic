Visual Music Command Line Parameters

Visual Music is planned to have several command line options which will provide the simplest method to invoke and control Visual Music. However in this edition, Visual Music supports only two parameters:

  • /RegIcon When passed this parameter to Visual Music, the MSC file extenstion is associated with Visual Music and it's icon is also registered with the MSC file extension. Visual Music exits after doing this registrations.
  • [FileName] When passed a file name as a parameter, Visual Music load it on start up.
