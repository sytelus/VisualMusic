TEMPO Statement

Slows down the execution by the factor of the specified value. It's kind of slow motion playing.

Syntax

TEMPO:<Slow down factor>

The TEMPO statement syntax has these arguments

Part Description
Slow down factor The factor with which system clock would be slowed down.

Short Form

None

Remarks

Default value is 1. This command slows down the whole script. Useful to inspect what's happening when notes are played very fast.

Examples


a a# b c# 'Normal speed

TEMPO:3

a a# b c# 'Now in 3 times lower speed
