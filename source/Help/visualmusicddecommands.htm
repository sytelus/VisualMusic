Visual Music DDE Commands

Visual Music is also a DDE server allowing other programs to talk with it using DDE link. For example, your application can detect the existing Visual Music instance by trying to set up a DDE link. Or even ask Visual Music to load a specific file and play it. The Visual Music itself uses it's own DDE capability to load the file in the existing instance when you click it in the Explorer. Below is the short description few DDE commands.

DDE Server Specifications:


LinkTopic: Visual Music|Main
LinkItem: BaseCommands

DDE Server Command List:


  • ACTIVATE Brings the currently running Visual Music instance in front of all windows.
  • EXIT Ends the currently running Visual Music instance.
  • PLAY([TuneName]) Plays the specified tune. If tune is not specified the it plays currently selected tune.
  • PLAY_ALLPlays all the tuns simultaneously.
  • STOP_PLAYStops any tunes that is currently being played.
  • LOAD(FileName) Loads the specified file.
  • SAVESaves the current tunes.
  • SAVE_AS(FileName) Saves the current tunes in to specified file.
  • NEWStarts new composition.
  • SHOW_HELP(HelpTopic) Shows specified help topic.
