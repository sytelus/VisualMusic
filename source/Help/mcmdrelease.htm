RELEASE Statement

What it does: Releases the MIDI channel for the use of other tunes.

Syntax

RELEASE

The RELEASE  statement syntax has no arguments


Short Form

None

Remarks

The MIDI device has only 16 channels in which you can play music. If you try to run more then 16 tunes simultaneously you will get "Out of channels" error (you can create more then 16 tunes but only 16 of them should be actively running at any time). The RELEASE instruction allows you to release the MIDI channel if you are not using it and hence there by use can run more tunes simultaneously. MIDI Engine incorporates smart channel pooling and automatic channel release during long silence periods so you should be almost never bother about using this instruction. Also note that there is no instruction for acquiring channels. Channel allocation is done completely automatically depending on which channel is free.

Examples

1.Write in Tune1 following:

'Say you are not playing any note here
'but only doing some arithmetic etc
'so release the channel so that others may use it.
RELEASE
---long processing---
a a# 'When you play notes next time, channel is automatically allocated
