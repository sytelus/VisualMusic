SILENCE Statement

Does nothing until specified interval is passed. It's essentially produces silence between two notes.

Syntax

SILENCE:<interval value>

The SILENCE statement syntax has these arguments

Part Description
interval value One interval is by default
equals to 60ms.

Short Form

S

Remarks

Default value is 1. You can also specify silence interval as the part of the note command.

Examples


a#
SILENCE:10
a#

2. 'Exactly same thing

a#
WAIT_FOR:10
a#

3. 'Exactly same thing

c#:,10
a#
