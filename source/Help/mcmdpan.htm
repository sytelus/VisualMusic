PAN Statement

Sets the difference of amount of loudness with which higher notes and lower notes are played. See What is Note, Octave and bla bla bla

Syntax

PAN:<Pan value>

The PAN statement syntax has these arguments

Part Description
Pan value Pan value

Short Form

None

Remarks

Default value is 64 which means all notes are played at equal volume.

Examples

1. Play notes in seperate pans

PAN:22
O:2 a#
O:6 a#
PAN:100
O:2 a#
O:6 a#

2. Play notes in decreasing pan

SET:Count,87
PAN:20
LABEL:TheLoop
NOTE:Count
MATH:Count,--
IF:Count,>,0,TheLoop
