Required Files

Following should be already installed on your machine before you can run Visual Music.

  1. Visual Basic 6.0 run time libraries.
  2. Windows common controls supplied with Microsoft Internet Explorer 5.0 or later.
  3. The hhctrl.ocx file to view compiled HTML help (.chm files) but this is not necessary.
Visual Music itself consist of following files:
  1. Visual Music.exe
  2. Instruments.ini
  3. VisualMusic.ico (not necessary)
  4. Visual Music.chm (not necessary)
  5. samples folder with several MSC sample files (non nessasary)
